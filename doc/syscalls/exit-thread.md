# EXIT_THREAD - Terminate the Current Thread

## Description

Terminate the current thread.

## Arguments

Function number (`arg0`) is 12.

```
    +----------------------------------------------------------------+
    |                        function = 12                           |  arg0
    +----------------------------------------------------------------+
    31                                                               0
    
    +----------------------------------------------------------------+
    |                         reserved (0)                           |  arg1
    +----------------------------------------------------------------+
    31                                                               0

    +----------------------------------------------------------------+
    |                         reserved (0)                           |  arg2
    +----------------------------------------------------------------+
    31                                                               0

    +----------------------------------------------------------------+
    |                         reserved (0)                           |  arg3
    +----------------------------------------------------------------+
    31                                                               0
```

## Return Value

This function terminates the current thread and does not return.

## Errors

This function always succeeds.
